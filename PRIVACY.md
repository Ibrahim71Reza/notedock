# Privacy Policy for NoteDock

NoteDock is a local-first browser extension.

## What NoteDock stores

NoteDock stores the following data locally on your device:

- notes
- note titles
- tags
- note formatting
- locally inserted images
- trash state
- favorites
- local backup files created by the user

## What NoteDock does not do

NoteDock does not:

- upload your notes to any server
- sell your data
- track your browsing history
- collect analytics
- require an account
- transmit your note content externally

## Permissions used

- Storage: to save notes locally
- Downloads: to export notes and backups
- ActiveTab: to capture the URL and page title when you right-click to clip web content
- Context Menus: to support web clipping from right-click actions
- Side Panel: to show the NoteDock sidebar UI
