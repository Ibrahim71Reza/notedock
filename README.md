# NoteDock — Sidebar Notes & Web Clipper

NoteDock is a lightweight browser sidebar extension for taking notes while browsing.

## Features

- Sidebar note editor
- Rich text formatting
- Inline images
- Drag and drop images
- Paste images from clipboard
- Tables with row/column add and delete
- Clickable links
- Tags
- Favorites
- Trash and restore
- Duplicate notes
- Search notes
- Export as NDock, TXT, HTML, Markdown, PDF
- Full workspace backup and restore
- Local-first autosave
- Resizable tools, editor, and notes sections

## Privacy

NoteDock stores your notes locally in browser storage and IndexedDB.
It does not send your notes to any external server.
